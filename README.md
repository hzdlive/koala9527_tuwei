用途是给js所写的游戏脚本的进行收费管理

## 详细功能介绍

服务端（php）介绍：

* 官网，用于展示、注册登录、忘记密码等；
* 后台网站，用户统计、软件管理、注册码管理、推荐人规则等功能；
* 供客户端调用的接口，请求数据使用rsa和des加解密，确保封包安全；

客户端（dll）介绍：

* 登录；
* 试用；
* 解绑；
* 离线；
* 推荐人加时；
